# 学生向け漢字練習ツール

このツールはブラウザだけで動く、漢字の「書き」と「読み」を練習できる学習アプリです。HTML1ファイルで完結しており、サーバーやAPIキーは不要です。

## 主な機能

### 📝 漢字リスト登録
- 1件ずつ、または「漢字,よみがな」形式でまとめて複数件を一括登録できます。
- 登録した漢字リストはブラウザのLocalStorageに保存され、次回アクセス時も保持されます。
- 「デフォルトを復元」ボタンで初期セットに戻すことも可能です。

### ✍️ 書き練習
- 出題されたよみがなに対応する漢字を、画面上のキャンバスに手書きで入力します。
- Googleの手書き文字認識エンジン（Gboardと同じ仕組み）を利用し、書いた文字をリアルタイムでAI判定。
- 1文字ずつのマス目ガイドが自動表示され、複数文字の単語にも対応します。

### ⌨️ 読み練習
- 表示された漢字の読み方をひらがなで入力して回答します。

### 🔥 弱点克服モード
- 書き練習・読み練習での誤答を自動的に記録。
- 間違いの多い漢字ほど出題されやすくなる「重み付きランダム出題」で、効率的に弱点を復習できます。

### ⚙️ 練習セッション設定
- 出題順（順番通り／完全ランダム）を選択可能。
- 1回の練習問題数（5〜30問、または全問）を指定できます。
- 練習終了後には結果サマリー（正答率など）を表示します。

## 使い方

1. 「漢字リスト登録」タブで練習したい漢字とよみがなを登録する
2. 「書き練習」または「読み練習」タブで出題範囲・問題数を設定して練習を開始する
3. 苦手な漢字が増えてきたら「弱点克服」タブで集中的に復習する

## 技術構成

- HTML / Tailwind CSS（CDN） / Font Awesome / バニラJavaScript
- データ保存：ブラウザのLocalStorage（サーバー不要）
- 手書き認識：Google手書き入力API（`inputtools.google.com`、Gboardが利用する非公式APIのため、将来仕様が変更される可能性があります）
